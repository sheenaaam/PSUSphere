# PSUSphere

## Short Description
PSUSphere is a centralized web application for managing student organizations, programs, and colleges. It allows students, organizations, and administrators to easily interact, track memberships, and manage data in a structured and user-friendly way.

## Features
- Manage Colleges, Programs, and Organizations
- Track Students and their memberships in organizations
- Customizable admin interface with search and filters
- Generate fake data for testing (using Faker)
- CRUD operations for all entities
